== Installation ==

*=== DEUTSCH ===*

1. [Kostenlos registrieren](http://www.jiffuy.com/registrieren "Kostenlos registrieren bei jiffuy") (DEUTSCH) für persönliche API-Zugangsdaten
2. Download und Installation des "jiffuy for Wordpress"-Plugins durch kopieren der Plugin-Dateien in das Verzeichnis "wp-content/plugins/jiffuy/" oder durch Verwendung der Wordpress Admin-Oberfläche
3. Aktivieren des "jiffuy for Wordpress"-Plugins durch hinzufügen der API-Zugangsdaten in der Wordpress Admin-Oberfläche
4. Geld verdienen mit der eigenen Website oder dem eigenen Blog!

*=== ENGLISH ===*

1. [Sign up for free](http://www.jiffuy.com/register "Sign up jiffuy") (ENGLISH) for an API key and secret
2. Download, Extract and upload the plugin by your Wordpress frontend or with any FTP client to the directory "wp-content/plugins/jiffuy/"
3. Activate the jiffuy for wordpress plugin in your admin panel with your api key and secret
4. Make money with your website or your Blog!
